# boid-simulation-opengl

## Dependencies

- CMake
- SDL2

## How to build

- `git clone https://github.com/SaferGo/boid-simulation-opengl.git`
- `cd boid-simulation-opengl`
- `mkdir build`
- `cd build`
- `cmake ..`
- `make`
- To run it: `./../bin/BoidSimulation`
